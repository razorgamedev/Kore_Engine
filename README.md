# Kore Engine
this is a simple game engine, written using c++ and modern OpenGL on Linux.

## Features...
  * Written in C++ 
  * Modern OpenGL using the programmable pipeline
  * 3D model loading
  * Texture mapping
  * Ambiant lighting
  * Point lights
  * It has a simple gui that you can use to edit the scene
  * First Person Camera view
  * Scene loading
